# DWP301c Cheatsheet

## Slot 3 - HTML

- HTML tao cau truc va noi dung cho trang web.
- Noi dung hien tren trang nam trong the `body`.
- `h1`, `h2` dung cho tieu de.
- `p` dung cho doan van.
- `ul` la danh sach khong thu tu, `ol` la danh sach co thu tu, `li` la tung muc.
- `a` dung de tao link, `img` dung de hien thi anh.
- Sua HTML xong thi luu file va refresh browser.

## Slot 4 - CSS

- CSS dung de trang tri giao dien.
- Gan CSS vao HTML bang `link rel="stylesheet" href="style.css"`.
- `background-color` doi mau nen.
- `color` doi mau chu.
- `padding` la khoang cach ben trong.
- `margin` la khoang cach ben ngoai.
- `border` tao vien, `border-radius` bo goc.
- `.class-name` chon cac phan tu co class tuong ung.
- `#id-name` chon phan tu co id tuong ung.
- `display: flex` giup sap xep cac phan tu tren cung mot hang linh hoat.
- `gap` tao khoang cach giua cac phan tu trong flex.

## Slot 5 - JavaScript va DOM

- JavaScript dung de tao tuong tac cho trang web.
- Gan JavaScript vao HTML bang `script src="script.js"`.
- DOM la cach JavaScript nhin va dieu khien cac phan tu HTML.
- `document.querySelector("#id")` tim mot phan tu trong HTML.
- `addEventListener("click", ...)` chay code khi nguoi dung bam.
- `classList.toggle("class-name")` bat/tat mot class tren phan tu.
- `event.preventDefault()` chan hanh vi mac dinh, vi du chan form reload trang.
