# Slot 6 - Git Demo

File nay duoc tao de thuc hanh luong Git co ban.

Luong can nho:

1. Sua hoac tao file.
2. Chay `git status` de xem thay doi.
3. Chay `git add <file>` de dua file vao staging area.
4. Chay `git commit -m "message"` de luu mot moc lich su.
5. Chay `git log --oneline` de xem lich su commit.
