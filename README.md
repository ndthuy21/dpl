# Car Dealership Application

Repository name: dpl
Project name: Car Dealership Application

This project is the IBM Full Stack Application Development Capstone Project. It implements a full-stack car dealership review portal with Django, React, Node.js, Express, MongoDB, Docker, Kubernetes, GitHub Actions, and IBM Cloud services.

## Features

- Static Home, About, and Contact pages.
- User registration, login, and logout.
- Dealership listing with state filtering.
- Dealer details page with customer reviews.
- Authenticated review submission.
- Express and MongoDB backend services for dealership and review data.
- Django proxy services for backend APIs.
- Sentiment analysis integration for review text.
- CI/CD workflow with GitHub Actions.
- Containerized deployment support.

## Technologies

- Python and Django
- JavaScript and React
- Node.js and Express
- MongoDB
